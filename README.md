# Cwarm
C++ based Particle Swarm Optimization

### Build instructions
The project is built using [Bazel](http://bazel.io/).

* Core library

    ```shell
    bazel build cwarm/core
    ```
* Tests

    ```shell
    bazel test tests/cwarm
    ```
