# Cswarm
C++ based Particle Swarm Optimization

### Build instructions
The project is built using [Bazel](http://bazel.io/).

* Core library

    ```shell
    bazel build cswarm/core
    ```
* Tests

    ```shell
    bazel test tests/cswarm
    ```
